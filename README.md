AI filmmaker, director, technical artist, and small-studio toolmaker based in Shanghai.

I make AI-native animated films and the production systems behind them: storyboards, prompt languages, model tests, local-first tools, agent workflows, and small creative interfaces.

My favorite creative shape is simple: **a tiny action opens a world**. A cat walks into a museum, a city, an artwork, or a public screen. The story feels light, but the pipeline underneath has to be precise.

## What I Build Here

- **Creative systems for AI filmmaking**: script expansion, text storyboards, contact sheets, panel splitting, shot-level prompts, delivery packages.
- **Director-style previs workflows**: fast ways to see motion, blocking, rhythm, and camera choreography before committing to final images.
- **Prompt formats as production documents**: prompts that preserve camera, motion, material, causality, edit rhythm, and art direction.
- **Local-first studio infrastructure**: project memory, reference boards, task logs, GitHub sync, private asset boundaries, and small automation tools.
- **Interfaces for invisible work**: playful frontends that make agent states, creative tasks, and long-running systems easier to understand.
